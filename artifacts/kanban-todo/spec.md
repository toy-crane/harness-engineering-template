## Overview
Trello 스타일의 칸반 보드 Todo 앱. 3칼럼 고정 보드에서 카드를 드래그&드롭으로 관리하고, 검색·필터로 원하는 카드를 빠르게 찾을 수 있다. 다크모드를 지원하며, 모든 데이터는 localStorage에 저장된다.

## 시나리오

### 1. 카드 추가
[상황] 사용자가 칼럼 하단의 '카드 추가' 버튼을 클릭하면
[동작] 제목 입력란이 나타나고, 제목을 입력 후 확인하면 해당 칼럼에 카드가 추가된다

성공 기준:
- [ ] 'To Do' 칼럼에서 '카드 추가' 클릭 → 제목 입력란 표시 → '장보기' 입력 후 확인 → 'To Do' 칼럼에 '장보기' 카드 표시
- [ ] 제목을 비워둔 채 확인 → '제목을 입력해주세요' 오류 메시지 표시

### 2. 카드 상세 편집
[상황] 사용자가 카드를 클릭하면
[동작] 카드 상세 모달이 열리고, 설명·우선순위·태그·서브태스크를 편집할 수 있다

성공 기준:
- [ ] '장보기' 카드 클릭 → 상세 모달에 제목, 설명, 우선순위, 태그, 서브태스크 섹션 표시
- [ ] 설명에 '마트 가서 장보기' 입력 후 저장 → 모달 닫고 다시 열면 '마트 가서 장보기' 표시
- [ ] 우선순위를 'High'로 변경 → 카드에 우선순위 표시 반영
- [ ] 태그 '개인' 입력 후 추가 → 카드에 '개인' 태그 표시

### 3. 카드 인라인 제목 편집
[상황] 사용자가 보드에서 카드 제목을 클릭하면
[동작] 제목이 인라인 편집 모드로 전환되어 바로 수정할 수 있다

성공 기준:
- [ ] '장보기' 카드 제목 클릭 → 입력란으로 전환 → '마트 장보기'로 수정 후 Enter → '마트 장보기'로 표시
- [ ] 빈 제목으로 수정 시도 → 원래 제목 '장보기' 유지

### 4. 카드 삭제
[상황] 사용자가 카드 상세 모달에서 '삭제' 버튼을 클릭하면
[동작] 확인 다이얼로그가 표시되고, 확인하면 카드가 삭제된다

성공 기준:
- [ ] 상세 모달에서 '삭제' 클릭 → '정말 삭제하시겠습니까?' 확인 다이얼로그 표시
- [ ] 확인 클릭 → 카드가 보드에서 제거됨
- [ ] 취소 클릭 → 카드 유지

### 5. 서브태스크 관리
[상황] 사용자가 카드 상세 모달에서 서브태스크를 추가하거나 체크하면
[동작] 체크리스트 형태로 서브태스크가 표시되고, 진행률이 업데이트된다

성공 기준:
- [ ] '서브태스크 추가' 입력란에 '우유 사기' 입력 후 추가 → 체크리스트에 '우유 사기' 표시
- [ ] '우유 사기' 체크박스 클릭 → 완료 표시, 진행률 '1/1' 표시
- [ ] 서브태스크 2개 중 1개 완료 → 진행률 '1/2' 표시

### 6. 드래그&드롭 칼럼 간 이동
[상황] 사용자가 카드를 드래그하여 다른 칼럼에 드롭하면
[동작] 카드가 해당 칼럼으로 이동한다

성공 기준:
- [ ] 'To Do'의 '장보기' 카드를 'In Progress'로 드래그 → 'In Progress' 칼럼에 '장보기' 표시, 'To Do'에서 제거
- [ ] 'In Progress'의 '장보기' 카드를 'Done'으로 드래그 → 'Done' 칼럼에 '장보기' 표시

### 7. 드래그&드롭 칼럼 내 순서 변경
[상황] 사용자가 같은 칼럼 내에서 카드를 드래그하여 다른 위치에 드롭하면
[동작] 카드 순서가 변경된다

성공 기준:
- [ ] 'To Do' 칼럼에 '장보기', '운동하기', '공부하기' 순서 → '공부하기'를 첫 번째로 드래그 → '공부하기', '장보기', '운동하기' 순서로 표시

### 8. 제목 검색
[상황] 사용자가 검색바에 키워드를 입력하면
[동작] 제목에 해당 키워드가 포함된 카드만 표시된다

성공 기준:
- [ ] '장보기', '공부하기', '운동하기' 카드 존재 → 검색바에 '하기' 입력 → 3개 카드 모두 표시
- [ ] 검색바에 '장' 입력 → '장보기' 카드만 표시, 나머지 숨김
- [ ] 검색어 삭제 → 모든 카드 다시 표시

### 9. 우선순위·태그 필터
[상황] 사용자가 필터 드롭다운에서 우선순위나 태그를 선택하면
[동작] 해당 조건에 맞는 카드만 표시된다

성공 기준:
- [ ] 우선순위 필터에서 'High' 선택 → 우선순위가 'High'인 카드만 표시
- [ ] 태그 필터에서 '개인' 선택 → '개인' 태그가 있는 카드만 표시
- [ ] 검색어 '장' + 우선순위 'High' → 두 조건 모두 만족하는 카드만 표시
- [ ] 필터 해제 → 모든 카드 다시 표시

### 10. 다크모드 토글
[상황] 사용자가 헤더의 다크모드 토글 버튼을 클릭하면
[동작] 앱 전체가 다크모드/라이트모드로 전환된다

성공 기준:
- [ ] 라이트모드에서 토글 클릭 → 배경이 어두운 색으로 변경
- [ ] 다크모드에서 토글 클릭 → 배경이 밝은 색으로 복원

### 11. localStorage 데이터 영속성
[상황] 사용자가 카드를 추가/수정/이동한 후 페이지를 새로고침하면
[동작] 이전 상태가 그대로 복원된다

성공 기준:
- [ ] 카드 3개 추가 후 새로고침 → 3개 카드 그대로 표시
- [ ] 카드를 'In Progress'로 이동 후 새로고침 → 'In Progress' 칼럼에 카드 유지
- [ ] 다크모드 설정 후 새로고침 → 다크모드 유지

## 범위

### 포함
- 고정 3칼럼(To Do / In Progress / Done) 칸반 보드
- 카드 CRUD (제목, 설명, 우선순위, 태그, 서브태스크)
- 카드 인라인 제목 편집
- 드래그&드롭 (칼럼 간 이동 + 칼럼 내 순서 변경)
- 제목 검색 + 우선순위·태그 필터 (AND 조합)
- 다크모드 토글
- localStorage 데이터 저장/복원

### 제외
- 사용자 인증/로그인: 로컬 전용 앱이므로 불필요
- 칼럼 커스터마이징(추가/삭제/이름변경): 고정 3칼럼으로 결정
- 마감일/담당자 필드: 기본 세트에서 제외
- 서버 동기화/멀티 디바이스: localStorage만 사용
- 카드 카테고리 분류: 태그로 대체
