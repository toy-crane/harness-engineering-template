# Trello 스타일 칸반 Todo 앱 - 요구사항 스펙

## Overview
개인 작업을 시각적으로 관리할 수 있는 Trello 스타일 칸반 보드. 카드를 드래그&드롭으로 칼럼 간 이동하고, 우선순위·태그로 필터링하며, 모든 데이터는 브라우저 localStorage에 저장된다.

## 시나리오

### 1. 카드 추가
[상황] 사용자가 Todo 칼럼에서 제목을 입력하고 추가하면
[동작] Todo 칼럼에 새 카드가 생성되고 localStorage에 저장된다

성공 기준:
- [ ] 제목 "새 작업" 입력 후 추가 → Todo 칼럼 마지막에 카드 생성, 기본 우선순위 Medium
- [ ] 빈 제목으로 추가 시도 → 카드 생성되지 않음, 에러 피드백 표시
- [ ] 공백만 있는 제목으로 추가 시도 → 카드 생성되지 않음
- [ ] 카드 추가 후 페이지 새로고침 → 추가한 카드가 그대로 존재

### 2. 카드 인라인 편집
[상황] 사용자가 기존 카드의 제목, 우선순위, 태그를 수정하면
[동작] 변경 사항이 즉시 반영되고 localStorage에 저장된다

성공 기준:
- [ ] 카드 제목 "기존 작업" → "수정된 작업"으로 변경 → 화면과 저장소 모두 반영
- [ ] 우선순위 Medium → High 변경 → 카드 색상이 High 색상(빨강 계열)으로 변경
- [ ] 태그 "디자인" 추가 → 카드에 태그 배지 표시
- [ ] 제목을 빈 문자열로 수정 시도 → 변경 거부, 이전 제목 유지

### 3. 카드 삭제
[상황] 사용자가 카드의 삭제 버튼을 클릭하면
[동작] 확인 다이얼로그가 표시되고, 확인 시 카드가 삭제된다

성공 기준:
- [ ] 삭제 버튼 클릭 → 확인 다이얼로그 표시
- [ ] 확인 다이얼로그에서 "확인" → 카드 삭제, localStorage 반영
- [ ] 확인 다이얼로그에서 "취소" → 카드 유지
- [ ] 삭제 후 새로고침 → 삭제된 카드가 복원되지 않음

### 4. 드래그 앤 드롭으로 칼럼 이동
[상황] 사용자가 카드를 드래그하여 다른 칼럼에 드롭하면
[동작] 카드가 대상 칼럼으로 이동하고 변경된 상태가 저장된다

성공 기준:
- [ ] Todo의 카드를 In Progress로 드래그 → 카드가 In Progress 칼럼에 표시
- [ ] In Progress의 카드를 Done으로 드래그 → 카드가 Done 칼럼에 표시
- [ ] 같은 칼럼 내에서 카드 순서 변경(재정렬) 가능
- [ ] 이동 후 새로고침 → 이동된 위치 유지

### 5. 검색 및 필터링
[상황] 사용자가 검색어를 입력하거나 필터를 선택하면
[동작] 조건에 맞는 카드만 실시간으로 표시되고 나머지는 숨겨진다

성공 기준:
- [ ] 검색어 "버그" 입력 → 제목에 "버그"가 포함된 카드만 즉시 표시
- [ ] 우선순위 "High" 필터 → High 카드만 표시
- [ ] 태그 "디자인" 필터 → 해당 태그를 가진 카드만 표시
- [ ] 검색어 + 우선순위 필터 동시 적용 → 두 조건 모두 만족하는 카드만 표시 (AND)
- [ ] 필터 해제 → 모든 카드 다시 표시
- [ ] 필터링은 실시간 (타이핑할 때 즉시 결과 반영, 별도 버튼 없음)

### 6. 다크모드 토글
[상황] 사용자가 다크모드 토글 버튼을 클릭하면
[동작] 테마가 전환되고 선택이 localStorage에 저장된다

성공 기준:
- [ ] 라이트 모드에서 토글 → 다크 모드 UI 적용
- [ ] 다크 모드에서 토글 → 라이트 모드 UI 적용
- [ ] 다크 모드에서 새로고침 → 다크 모드 유지
- [ ] 첫 방문 시 → 라이트 모드 기본값

## 전제 조건
- Next.js (App Router) + TypeScript 프로젝트
- shadcn/ui 컴포넌트 라이브러리 사용
- Vitest + React Testing Library로 테스트
- 서버/DB 없이 localStorage만 사용
- 칼럼은 Todo / In Progress / Done 3개 고정 (사용자 추가/삭제 불가)

## 결정 사항
- 카드 삭제 시: 확인 다이얼로그 표시 (실수 방지)
- 태그 입력 방식: 자유 입력 (사용자가 텍스트 직접 타이핑)
- 검색 방식: 실시간 필터 (타이핑 즉시 반영)
- 같은 칼럼 내 카드 순서 변경: 지원
- 우선순위 색상: High=빨강 계열, Medium=노랑 계열, Low=초록 계열
- 첫 방문 기본 테마: 라이트 모드
