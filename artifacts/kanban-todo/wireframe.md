## Screen: 기본 보드 (scenarios: 1, 2, 4, 5, 6)

### Layout Structure

```
Screen: 기본 보드 [col]
├── Header [row] — 앱 타이틀(중앙), 다크모드 토글(우측)
├── Toolbar [row] — 검색 input(넓게), 우선순위 필터, 태그 필터
└── Board [row, 1:1:1]
    ├── Column: Todo [col] — 칼럼 헤더 + 카드 목록 + 카드 추가 입력
    ├── Column: In Progress [col] — 칼럼 헤더 + 카드 목록
    └── Column: Done [col] — 칼럼 헤더 + 카드 목록
```

### Area: Board

> 칼럼 3개는 동일 구조. Todo만 하단에 카드 추가 입력 포함. 대표로 Todo를 상세 표시.

```
┌─── ⑥ Todo (3) ────────┐  ┌── In Progress (1) ────┐  ┌──── Done (2) ─────────┐
│                        │  │                        │  │                        │
│ ┌────────────────────┐ │  │ ┌────────────────────┐ │  │ ┌────────────────────┐ │
│ │ API 버그 수정   🗑 │ │  │ │ DB 마이그레이션 🗑 │ │  │ │ 로고 디자인     🗑 │ │
│ │ (● High) (디자인)  │ │  │ │ (● Medium) (개발)  │ │  │ │ (● Low) (디자인)   │ │
│ └────────────────────┘ │  │ └────────────────────┘ │  │ └────────────────────┘ │
│ ┌────────────────────┐ │  │                        │  │ ┌────────────────────┐ │

│ │ (● Medium) (개발)  │ │  │                        │  │ │ (● Medium) (개발)  │ │
│ └────────────────────┘ │  │                        │  │ └────────────────────┘ │
│ ┌────────────────────┐ │  │                        │  │                        │
│ │ 테스트 추가     🗑 │ │  │                        │  │                        │
│ │ (● Low)            │ │  │                        │  │                        │
│ └────────────────────┘ │  │                        │  │                        │
│                        │  │                        │  │                        │
│ ┌────────────────────┐ │  │                        │  │                        │
│ │ ⑩ 새 작업 입력... │ │  │                        │  │                        │
│ └────────────────────┘ │  │                        │  │                        │
└────────────────────────┘  └────────────────────────┘  └────────────────────────┘
         ⑦ ⑧ ⑨
```

카드 상태:
- 편집 모드: 카드 클릭 시 제목이 편집 가능한 입력으로 전환 `(← inline edit)`
- 드래그 중: 카드가 반투명 + 드롭 대상 칼럼 하이라이트 `(← drag state)`
- 필터 적용 시: 조건 불일치 카드 숨김 `(← filtered)`

### Components

| # | Name | Type | Detail |
|---|------|------|--------|
| ① | 앱 타이틀 | Text | "Kanban Todo", 중앙 정렬 |
| ② | 다크모드 토글 | Toggle | [●━━━] on=dark, [━━━●] off=light, 우측 끝 배치. ☀/🌙 아이콘 |
| ③ | 검색 입력 | Input | placeholder "검색...", 실시간 필터링. 넓은 비율 |
| ④ | 우선순위 필터 | Select | options: All / High / Medium / Low |
| ⑤ | 태그 필터 | Select | options: All / 사용 중인 태그 목록 (동적) |
| ⑥ | 칼럼 헤더 | Text | 칼럼명 + 카드 수 카운트 |
| ⑦ | 카드 | Card | 제목 + 삭제 아이콘 + 우선순위 배지 + 태그 배지. 드래그 가능 |
| ⑧ | 우선순위 배지 | Badge | (● High)=빨강, (● Medium)=노랑, (● Low)=초록 |
| ⑨ | 태그 배지 | Badge | (태그명) 형태, 복수 가능 |
| ⑩ | 카드 추가 입력 | Input | Todo 칼럼 하단. placeholder "새 작업 입력...", Enter로 추가 |

### Interactions

- [②] 토글 클릭 → 라이트/다크 모드 전환, localStorage에 테마 저장
- [③] 검색어 타이핑 → 제목에 검색어 포함된 카드만 실시간 표시
- [④] 우선순위 선택 → 해당 우선순위 카드만 표시 (검색과 AND 조합)
- [⑤] 태그 선택 → 해당 태그를 가진 카드만 표시 (검색, 우선순위와 AND 조합)
- [⑦] 카드 클릭 → 제목 인라인 편집 모드 전환
- [⑦] 카드 드래그 → 다른 칼럼으로 이동 또는 같은 칼럼 내 순서 변경
- [⑦] 🗑 클릭 → 삭제 확인 모달 표시
- [⑧] 우선순위 배지 클릭 → 우선순위 변경 (High → Medium → Low 순환 또는 드롭다운)
- [⑨] 태그 영역 클릭 → 태그 추가/삭제 입력
- [⑩] Enter 키 → 입력된 제목으로 새 카드 생성 (빈 제목 시 무시)

### Components

| # | Name | Type | Detail |
|---|------|------|--------|
| ① | 다이얼로그 제목 | Text | "카드를 삭제할까요?" |

- Esc 키 → 모달 닫기 (취소와 동일)
