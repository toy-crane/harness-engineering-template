## Overview
웹 아티클이나 YouTube 영상의 URL을 입력하면 콘텐츠를 추출하여 미리보기로 보여주고, AI 챗봇(ChatGPT, Claude)으로 바로 보낼 수 있는 미니멀 도구

## 시나리오

### 1. 아티클 URL로 콘텐츠 추출
[상황] 사용자가 URL 입력란에 웹 아티클 URL을 입력하면
[동작] 시스템이 defuddle로 본문과 메타데이터를 추출하여 미리보기를 표시한다

성공 기준:
- [ ] URL 입력란에 `https://example.com/article` 입력 후 제출 -> 제목, 저자, 발행일, 사이트명이 상단에 표시되고 본문 미리보기가 아래에 표시된다
- [ ] 추출 중 로딩 상태가 표시된다

### 2. YouTube URL로 transcript 추출
[상황] 사용자가 URL 입력란에 YouTube URL을 입력하면
[동작] 시스템이 defuddle로 transcript와 메타데이터를 추출하여 미리보기를 표시한다

성공 기준:
- [ ] URL 입력란에 `https://www.youtube.com/watch?v=dQw4w9WgXcQ` 입력 후 제출 -> 영상 제목이 상단에 표시되고 transcript 미리보기가 아래에 표시된다
- [ ] 추출 중 로딩 상태가 표시된다

### 3. 추출된 콘텐츠를 AI 챗봇으로 보내기
[상황] 콘텐츠가 추출되어 미리보기가 표시된 상태에서
[동작] 사용자가 Open In Chat 버튼을 클릭하면 ChatGPT 또는 Claude를 선택하여 추출된 원문을 해당 챗봇으로 전달한다

성공 기준:
- [ ] 미리보기 표시 상태에서 Open In Chat 드롭다운 클릭 -> ChatGPT, Claude 두 개 옵션이 표시된다
- [ ] ChatGPT 선택 -> 추출된 원문 텍스트가 ChatGPT 채팅으로 전달된다
- [ ] Claude 선택 -> 추출된 원문 텍스트가 Claude 채팅으로 전달된다

### 4. 잘못된 URL 입력 시 오류 표시
[상황] 사용자가 올바르지 않은 URL을 입력하면
[동작] 시스템이 오류 메시지를 표시한다

성공 기준:
- [ ] URL 입력란에 `not-a-url` 입력 후 제출 -> "올바른 URL을 입력해주세요" 오류 메시지가 표시된다

### 5. 콘텐츠 추출 실패 시 오류 표시
[상황] 유효한 URL이지만 콘텐츠를 추출할 수 없는 경우
[동작] 시스템이 오류 메시지를 표시한다

성공 기준:
- [ ] 접근 불가능한 URL 제출 -> "콘텐츠를 추출할 수 없습니다" 오류 메시지가 표시된다

## 범위

### 포함
- URL 입력 폼
- defuddle 기반 콘텐츠 추출 (아티클 + YouTube)
- 메타데이터(제목, 저자, 발행일, 사이트명) + 본문 미리보기
- Open In Chat으로 ChatGPT, Claude에 원문 전달
- URL 유효성 검증 및 추출 실패 오류 처리
- 추출 중 로딩 상태

### 제외
- 콘텐츠 편집 기능 — 미니멀 원칙, 추출 결과를 그대로 사용
- 기본 프롬프트 추가 — 대화는 AI 챗봇에서 자유롭게
- ChatGPT, Claude 외 다른 AI 챗봇 — 핵심 두 개로 시작
- 추출 히스토리 저장 — 1회성 도구 성격
- 브라우저 익스텐션 — URL 직접 입력 방식만

## 전제 조건
- Next.js 프로젝트가 세팅되어 있다
- defuddle 패키지를 설치할 수 있다
